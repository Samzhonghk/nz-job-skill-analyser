# NZ Job Skill Analyser

A simple Python tool that analyses job descriptions and counts common technical skills.

## What It Does

This project reads a job description from a text file and checks how many times selected technical skills appear.

It currently supports:

- Python
- SQL
- Java
- Docker
- AWS
- PostgreSQL

## Why I Built This

I built this project to practise Python fundamentals, including:

- Reading text files
- Functions
- Loops
- Dictionaries
- Conditional logic
- Type hints
- Regular expressions
